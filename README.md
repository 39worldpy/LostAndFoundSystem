#Campus Lost & Found System

##Project Overview
The Campus Lost & Found System is a web-based application designed for QIU students to report and track lost and found items within the university campus. 

The system digitalizes the traditional manual process (notice boards and WhatsApp groups) into a centralized, searchable platform.

---

## Features
### Student Features
- Register using QIU email
- Secure login authentication
- Submit lost item report
- Submit found item report
- Upload item image
- Search and filter items by name, category, and status
- View personal submission history
- Update report status
- Logout functionality

Run Locally (Full System)

To run the complete system with backend and database:

Install dependencies

npm install

Start the server

node server.js

Open browser and visit

http://localhost:3000/index.html

Make sure MySQL (XAMPP) is running before starting the server.
